Git基础命令学习

发布日期：2024-01-20

1. 仓库初始化

使用 git init 命令初始化一个新的Git仓库。这会创建一个.git目录来存储仓库信息。

git init

2. 文件暂存

使用 git add 命令将文件添加到暂存区，为提交做准备。

git add 文件名    # 添加单个文件
git add .        # 添加所有文件

3. 提交更改

使用 git commit 命令将暂存的更改保存到仓库中。

git commit -m "提交说明"

4. 查看状态

使用 git status 命令查看仓库的当前状态，包括哪些文件被修改、哪些文件在暂存区等。

git status

5. 查看历史

使用 git log 命令查看提交历史记录。

git log          # 详细历史
git log --oneline # 简洁历史

6. 比较差异

使用 git diff 命令查看文件的具体更改。

git diff              # 查看未暂存的更改
git diff --staged      # 查看已暂存的更改
git diff commit1 commit2  # 比较两个提交之间的差异

7. 撤销更改

Git提供了多种撤销更改的方式。

git restore 文件名     # 撤销工作区的修改
git restore --staged 文件名  # 撤销暂存区的修改
git reset --hard HEAD^  # 撤销最近一次提交

注意：使用 reset --hard 时要小心，它会永久删除更改！

8. 查看文件历史

查看特定文��的修改历史。

git log -p 文件名     # 查看文件的修改历史
git blame 文件名     # 查看文件的每一行是谁修改的

# 我的个人博客项目

这是一个用于学习 Git 的个人博客项目。通过实际开发这个博客系统，我们将学习 Git 的各种功能和最佳实践。

## 功能特点

- 响应式设计
- 深色/浅色主题切换
- 文章管理
- 关于页面
- 联系方式

## 项目结构

```
my-blog/
├── index.html          # 主页
├── about.html         # 关于页面
├── styles.css         # 样式文件
├── articles/          # 文章目录
│   └── first-post.html  # Git学习笔记
└── README.md          # 项目说明
```

## 技术栈

- HTML5
- CSS3
- JavaScript
- Git

## 本地开发

1. 克隆项目：
```bash
git clone https://github.com/你的用户名/my-blog.git
```

2. 打开 index.html 即可预览
